Q - How do the elements appear? Does the width have any effect? Why / Why not?

Elements appear side-by-side. As span itself is an inline tag, widht does not have any effect.

Q - Set display to inline. Does it cause any change? Why / Why not?

No. It deos not cause any because bydefault span is an inline tag.

Q - Change display to block. What change does it cause?

Once we apply block property span (bydefault inline) acts as a block element and takes up the entire width (if width not specified) by blocking every other element to be in same row.

Q - Set display to inline-block. What change does it cause?

Elements appear side-by-side with given width.

Conclusion:

inline: Elements are placed side-by-side but we don't have control over the width.
block: Takes up the entire width and blocks everything else from being on the same line. While using block property we can specify the width.
inline-block: In this case we can change width while letting elements to be side-by-side.
